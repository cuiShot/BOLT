Dataset description:
The puma32H dataset (OpenML data id 308) belongs to the Pumadyn family, synthetically generated from a realistic simulation of the Unimation PUMA 560 robot arm. Each record corresponds to a simulated robot state. The learning objective is to predict the angular acceleration of joint 6 from joint angles, angular velocities, applied torques, and additional dynamic parameters. The dataset contains 8,192 instances and 33 columns in total (32 continuous input features plus one continuous target). There are no missing values. In the original DELVE release, the cases are often referenced as 4,500 training and 3,692 test examples; OpenML typically provides the full combined table. The “H” in the name indicates a high-noise variant; with 32 inputs this variant is relatively linear but noisy.
